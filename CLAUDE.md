# claude-plugins

See README.md for what this repo is, the list of plugins, and how the marketplace works.

When adding, renaming, or removing a plugin, update all of these together:

- `.claude-plugin/marketplace.json`
- The "Plugins" table in README.md
- The "Install what you need" section in README.md — it must list every available plugin, one `/plugin install <name>@crafter-ai` line each, so users can copy&paste any of them
